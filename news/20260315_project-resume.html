BVE 真岡線 Project 活動再開のお知らせ

BVE真岡線 2026-03-15

平素より BVE 真岡線 Project を応援いただき、誠にありがとうございます。
長らくの休止期間を含め、プレイヤー・ファンの皆様には多大なるご理解とご支援を賜りましたこと、心より御礼申し上げます。
BVE 真岡線 Project は、2025年5月より活動を一時休止しておりましたが、 このたび再開の目処がつきましたのでお知らせいたします。

1. 活動再開日程

2026年4月より順次、BVEデータの制作を含む休止していた活動を再開いたします。

2. SNSアカウントの運用に関して

これまで、BVE 真岡線 Project のXでは、BVEの制作状況他、制作者の日常的な発信などをまとめてを行っていましたが、運営・制作者の個人アカウントを作成しました。
今後、Projectのアカウント(@Cottonway_PJ)では、BVEの進捗報告を毎週日曜に行います。また、せきたんのアカウント(@sekitan_ofc)にて日常的な発信などを行う予定です。
※Projectアカウントで行うフォローについては、これまで創作活動等にご協力いただいた方を除き、フォローを整理する場合がございます。予めご了承ください。

その他SNSのアカウントは、Projectのサイトにてご確認ください。

3. ProjectのHP開設について

以前、BVE真岡線Webの名前で公開されていたサイトがございましたが、この度の活動再開に際して、全く新しくサイトを公開いたしました。
サイトのURLは、https://sekitan-ofc.github.io/cottonway-project.html です。
今後、制作の進捗や公開予定のデータについての情報を随時公開していきます。

4. Project ロゴのバリエーション追加について

これまでのロゴに加え、BVE 真岡線 Project のロゴを新たに作成しました。
今後、従来のロゴに加えて順次使用されます。

5. その他

休止前に公開していたデータについて、再公開の予定はございません。また、当面の間、データの公開は行わない予定です。SNSで制作の過程を発信いたしますのでお楽しみください。
本プロジェクトは、当初より一貫して制作者1名の体制で運営されてまいりました。今後につきましても、同様に1名体制での活動を継続していく方針です。そのため、Projectメンバーの募集は行わない予定です。ただし、技術的な面において、個別にご協力をお願いする場合がございます。
真岡線に関する資料の提供等は、随時受け付けております。
今後とも本Projectの活動をあたたかく見守っていただけますと幸いです。

今後ともどうぞよろしくお願いいたします。

ーBVE 真岡線 Project
